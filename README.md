## Robinood-Clone
Robinhood clone app using react and firebase

<!-- <strong>Application live link:</strong> https://covid-19-tracker-2-1b97e.web.app/ -->

**Download**

`> git clone https://github.com/harvi7/robinhood-clone.git`

In your terminal write:

`> cd robinhood-clone`

And run:

`> npm install`
`> npm run`
