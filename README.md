## Robinood-Clone
Robinhood clone app using React, Redux and Firebase

<strong>Application live link:</strong> https://robinhood-clone-34795.web.app

**Download**

`> git clone https://github.com/harvi7/robinhood-clone.git`

In your terminal write:

`> cd robinhood-clone`

And run:

`> npm install`
`> npm run`
